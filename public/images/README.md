# Images Folder

Store your images here to use in your application.

## Usage

Place your images in this folder and reference them in your Vue components like:

```vue
<!-- Example: Using an image from this folder -->
<img src="/images/your-image.png" alt="Description" />
```

## Folder Structure

You can organize images in subfolders:
- `/images/products/` - Product images
- `/images/categories/` - Category images  
- `/images/banners/` - Banner images
- etc.

All files in the public folder are served at the root path.
